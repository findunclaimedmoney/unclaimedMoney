## Render Deployment — Full Migration Instructions

### Step 1: Push code to GitHub

The full monorepo needs to be in a GitHub repo. Render deploys from GitHub.

If not already on GitHub:
```bash
git remote add origin https://github.com/YOUR_USERNAME/missingcash.git
git push -u origin main
```

### Step 2: Create Render services

You need two services on Render:

**A) PostgreSQL Database**
- Render dashboard → New → PostgreSQL
- Note down both:
  - **Internal Database URL** (use in app env vars)
  - **External Database URL** (use to restore data from your laptop)

**B) Web Service (API Server)**
- Render dashboard → New → Web Service
- Connect your GitHub repo
- Settings:
  - **Root Directory**: *(leave blank)*
  - **Build Command**: `pnpm install && pnpm --filter @workspace/api-server run build`
  - **Start Command**: `pnpm --filter @workspace/api-server run start`
  - **Environment**: Node

### Step 3: Set environment variables on Render

In the Web Service → Environment → Add the following:

| Key | Value |
|-----|-------|
| `DATABASE_URL` | Internal URL from Render Postgres |
| `OPENAI_API_KEY` | (from Replit secrets) |
| `ELEVENLABS_API_KEY` | (from Replit secrets) |
| `ELEVENLABS_VOICE_ID` | `x3PfG9wL6FOEApZ1VJ9H` |
| `RESEND_API_KEY` | (from Replit secrets) |
| `STRIPE_SECRET_KEY` | (from Replit secrets) |
| `STRIPE_WEBHOOK_SECRET` | (from Replit secrets) |
| `MISSINGCASH_DOMAIN_VERIFIED` | `true` (only after missingcash.com.au verified in Resend) |

### Step 4: Restore the database

On your local machine, with the downloaded `missingcash_export.sql.gz`:

```bash
gunzip -c missingcash_export.sql.gz | psql YOUR_RENDER_EXTERNAL_URL
```

This copies all existing prospects, paid searches, crawl progress, and all other data.

### Step 5: Deploy

Render will auto-deploy when you push to GitHub. First deploy will:
1. Install pnpm dependencies
2. Build the API server
3. Start on the assigned port

### Step 6: Update Stripe webhook

In Stripe dashboard → Webhooks → update the endpoint URL from the Replit domain to your Render domain:
```
https://YOUR_RENDER_DOMAIN/api/stripe/webhook
```
