# MissingCash

Australian "unclaimed money" search landing site. Freemium funnel: search animation →
paywalled results → Stripe checkout for PDF guides.

Pure static HTML/CSS/vanilla JS. No backend, no build step. GA4 + external Stripe
checkout links only. Serve any of the `.html` files from any static host.

## Source of truth

Consolidated from `Downloads/missingcash-FINAL-COMPLETE.zip` (dated 2026-05-17, the
newest and most complete of ~6 zip variants and 7+ loose HTML files). Older variants
(`missingcash-v10.html`, `missingcash-DEPLOY_1/`, `missingcash-COMPLETE*.zip`, etc.)
are superseded and can be deleted from Downloads.

## Pages

| File | Purpose |
|------|---------|
| `index.html` | Main landing page / search funnel |
| `crypto.html` | Crypto recovery upsell |
| `cyber-security.html` | Cyber-security guide product page |
| `identity-theft.html` | Identity-theft guide product page |
| `bundle.html` | Bundle offer |
| `thankyou-*.html` | Post-purchase confirmation pages (one per product) |
| `*.pdf` | The guide PDFs delivered after purchase |

## Run locally

```sh
python -m http.server 8088
# then open http://localhost:8088/
```
